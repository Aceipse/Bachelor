  • requestWithResponse(action:type:body:completionHandler:)

    Sends a HTTP request to the WisRApi with the specified action and body. Then runs a callback-function with the received data as parameters. - parameter action: Is the HTTP call to invoke on the WisRApi. Follows the /Controller/Action pattern. - parameter type: Type of call: POST or GET. - parameter body: The body of the POST, empty if GET. - parameter completionHandler: A closure function to be run when the HTTP request has completed and the client has received an answer from the WisRApi. - parameter notification: Data returned from the WisRApi is always a JSON string of Notification. This function then parses the JSON string to this Notification class. - parameter response: Metadata associated with the response. See Apple NSURLResponse documentation. - parameter error: Error returned from the server. See Apple dataTaskWithRequest documentation.
