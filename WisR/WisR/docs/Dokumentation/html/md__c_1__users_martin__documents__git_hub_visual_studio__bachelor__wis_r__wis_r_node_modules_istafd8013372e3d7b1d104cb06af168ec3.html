which

Like the unix which utility.

Finds the first instance of a specified executable in the PATH environment variable. Does not cache the results, so hash -r is not needed when the PATH changes.

USAGE

1 var which = require('which')
2 
3 // async usage
4 which('node', function (er, resolvedPath) {
5  // er is returned if no "node" is found on the PATH
6  // if it is found, then the absolute path to the exec is returned
7 })
