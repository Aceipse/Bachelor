Use this instead of `require(\textquotesingle{}os\textquotesingle{}).tmpdir()` to get a consistent behaviour on different Node.\+js versions (even 0.\+8).

{\itshape This is actually taken from io.\+js 2.\+0.\+2 as it contains some fixes that haven\textquotesingle{}t bubbled up to Node.\+js yet.}
