Votes
filter in module dashboard

Description

Filter that counts the amount of votes of specified value (-1 == Downote, 1 == Upvote)

Usage

In HTML Template Binding

{{ Votes_expression | Votes:value }}

In JavaScript

$filter('Votes')(votes, value)

Parameters

ParamTypeDetails
votesArray<Vote>

Array with votes to be filtered

valueInteger

The value to check for(-1 or 1)

Returns

Integer

counter the filtered ammount of votes of the specified value
