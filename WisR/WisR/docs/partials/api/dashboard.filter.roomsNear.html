roomsNear
filter in module dashboard

Description

Filter that returns the rooms that are nearby(based on location)

Usage

In HTML Template Binding

{{ roomsNear_expression | roomsNear:scope }}

In JavaScript

$filter('roomsNear')(rooms, scope)

Parameters

ParamTypeDetails
roomsArray<Room>

the array of rooms to be filtered

scopeScope

the angular scope for the controller(so that we can access the currentLocation of the user)

Returns

Array<Room>

filtered the filtered array of rooms
